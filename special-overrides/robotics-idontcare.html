Meet the robot

The 2017 Robot


Introduction

The various pictures below shows a panoramic overview of La Salle Robotics' 2017 robot, "Albie". The robot is designed to compete in the 2017 FIRST Robotics Competition.

Design

The 2017 Robot "Albie" features a steam-themed industrial revolution-era design. The design is done according to the FIRST Robotics Competition (FRC)'s 2017 game theme "SteamWorks". The game features a rectangular playing ground with multiple robots divided into a red and a blue team, competing against each other. Industrial Revolution-era England is revisited with a steam-focused theme, leading the game to have high chimneys, towers, gears and balls for the robot to score points. On the 2017 Robot "Albie", the chasis is spray-painted to a metallic color to echo the theme for the 2017 Robotics Competition.

Build

Each year the robot is built by a team of mentors and students, using various different components. The 2017 Robot "Albie" features an aluminium chasis, bumpers, a ball intake/shooter mechanism, a gear-transport mechanism, and a climber. The robot is designed to be strictly compliant with FIRST's 2017 game "steamworks", as each of its components are made to make the robot score points in the SteamWorks game.
